## Zellij

Configured in `modules/home/zellij.nix` (default mode: `locked`, toggle with `Ctrl + g`).

| Key | Action |
|---|---|
| `Alt + t` | New tab |
| `Alt + w` | Close tab |
| `Alt + a` | Next tab |
| `Alt + 1..5` | Go to tab 1..5 |
| `Alt + h/j/k/l` | Move focus (left/down/up/right) |
| `Alt + f` | Toggle floating panes |
